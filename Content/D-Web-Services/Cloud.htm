Cloud

In the sky, a cloud is a collection of tiny water droplets or ice crystals suspended in the air. In IT, ‘cloud’ refers to cloud computing where services and storage are provided over the internet, enabling users to access data and applications from any device that has internet access.

The cloud in IT refers to servers that are accessed over the Internet, and the software and databases that run on those servers. Cloud servers are located in data centers all over the world. By using cloud computing, users and companies don’t have to manage physical servers themselves or run software applications on their own machines.Cloud computing is a type of computing that relies on shared computing resources rather than having local servers or personal devices to handle applications.

When working with cloud, it’s important to understand the following :

  • The cloud is a distributed collection of servers that host software and infrastructure.
  • It is accessed over the Internet.
  • Cloud servers are located in data centers all over the world.
  • Cloud computing allows users and companies to not have to manage physical servers themselves or run software applications on their own machines.
  • The cloud enables users to access the same files and applications from almost any device.

The five tasks cloud serves for:

  1. Storing data remotely.
  2. Accessing software over the internet.
  3. Scaling computing resources on demand.
  4. Hosting websites and applications.
  5. Analyzing big data.

Cloud limitations

Cloud computing provides scalable, on-demand resources, but it’s not without limitations. It requires a reliable internet connection, can have higher long-term costs, and raises concerns about data security and privacy.

Cloud lifespan

Cloud's lifespan” is about its ability to consistently run software across various environments. The durability of data in the cloud is often measured in “nines”. For instance, Amazon S3 and Google Cloud Platform claim 11 nines, or 99.999999999% annual durability, meaning if you store 1 million objects for 10 million years, you’d expect to lose 1 file. Durability can vary, so regular monitoring and maintenance are recommended.
